### 1.建库
在MySQL执行项目中的 `sql.sql` 建库

线上部署的记得将mysql改成utf8

### 2.配置阿里云OSS
在项目中的 `application.yml` 配置

线上部署的配置`application-prod.yml`

不太清楚的，可以进入 **阿里云** 中找一下 **对象存储OSS**
如果需要使用别的云，可以自行修改源码，主要修改项目中的 `OSSService.java` 。

### 3.配置数据库
在项目中的 `application.yml` 配置
